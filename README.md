# Tasty Restaurant Website

A modern and responsive restaurant landing page built with **HTML, CSS and JavaScript**.  
This project focuses on clean UI design, smooth user experience and interactive elements for a small restaurant concept.

## Features

- Responsive design (mobile & desktop compatible)
- Dynamic food cards created with JavaScript
- Order button with hover effects
- Food images and price tags
- Smooth scroll animations
- Social media icons with hover interactions
- Logo and styled header
- Contact section with email and phone info
- Simple search/filter feature for menu items
- Clean and minimal UI with blur and glass effects

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## Project Goal

The goal of this project is to demonstrate front-end development skills such as layout design, DOM manipulation, responsiveness and user interface details.  
It is designed as a portfolio project for junior developer level.
